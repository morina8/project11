About

Our goal is to make reading and study space easy to access.

Brookdale Public Library is a fictional library website created for a basic web technology assignment. The project keeps the structure simple and organized while still looking complete and functional.

Mission

The library offers a welcoming place for reading, learning, and community events. Visitors can browse books, find a quiet place to study, and ask for help at the front desk.

Services

  • Borrow books, magazines, and audiobooks
  • Use computers and free Wi-Fi
  • Reserve a study area for short sessions
  • Join book clubs and reading events

Simple Layout

Each page uses the same navigation bar and the same stylesheet for a consistent appearance.

Clean Structure

The project uses a root folder with three HTML files, one CSS folder, and one images folder.

Basic Styling

The design uses readable spacing, cards, buttons, and responsive sections without adding unnecessary complexity.
