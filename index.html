About

A simple library website made to feel welcoming, useful, and easy to use.

Brookdale Public Library is a fictional website created for a web design assignment. The goal of this project was to build something clean, organized, and realistic, without turning it into a spaceship dashboard for books.

Mission

The library is designed to feel like a comfortable place for reading, studying, and getting a little peace and quiet. It gives visitors a place to find books, use helpful resources, and take a break from the usual noise.

Services

  • Borrow books, magazines, and audiobooks
  • Use computers and free Wi-Fi
  • Reserve a study space for short sessions
  • Join reading groups and community events

Consistent Layout

Each page uses the same navigation and overall style, so the website feels connected instead of stitched together at the last second.

Organized Files

The project includes separate HTML pages, a CSS folder, and an images folder to keep everything simple and easy to manage.

Readable Design

The layout focuses on spacing, contrast, and clear sections so the content is easy to follow, which is kind of the whole point.
